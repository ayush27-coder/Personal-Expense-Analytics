# Personal Expense Analysis

## Project Overview
This project focuses on analyzing personal spending habits using Tableau. The dashboard provides insights into expense patterns, spending categories, payment methods, and overall financial behavior.

## Objective
The main objective of this project is to:
- Track personal expenses.
- Analyze spending across different categories.
- Monitor monthly expenditure trends.
- Identify preferred payment methods.
- Support better budgeting and financial planning.

## Tools Used
- Tableau
- Microsoft Excel
- GitHub

## Dataset Information
The dataset contains personal expense records with the following fields:

- Date
- Category
- Amount
- Payment_Mode

### Categories Included
- Food
- Transport
- Shopping
- Entertainment
- Bills
- Healthcare
- Education

### Payment Modes
- UPI
- Cash
- Card
- Net Banking

## Key Performance Indicators (KPIs)
- Total Expenses
- Average Expense
- Highest Expense
- Total Transactions

## Dashboard Features

### 1. Monthly Expense Trend
Visualizes spending patterns over time.

### 2. Category-wise Expense Analysis
Shows how expenses are distributed across different categories.

### 3. Payment Mode Analysis
Displays the usage of various payment methods.

### 4. Top Spending Categories
Highlights categories with the highest expenditure.

### 5. Expense Distribution
Provides insights into transaction values and spending behavior.

## Key Insights
- Shopping and Food contribute significantly to overall expenses.
- UPI is one of the most commonly used payment methods.
- Monthly spending varies based on purchasing behavior.
- Bills and Healthcare represent recurring essential expenses.

## Conclusion
The Personal Expense Analysis Dashboard helps users understand their spending habits and make informed financial decisions. The dashboard offers interactive visualizations and meaningful insights for effective expense management.
